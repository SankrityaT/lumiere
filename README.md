# Lumière

An editorial AI assistant interface — a frontend showcase rivaling the best chatbot UIs.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## What's in here

- **Two themes** — Cinematic Editorial Dark (warm black + honey amber) and Parchment Atelier Light (cream + ink). Toggle in the sidebar footer.
- **Distinctive typography** — Fraunces (variable serif display), Geist (body), Instrument Serif (italic accent), JetBrains Mono (UI metadata + code).
- **Full AI response anatomy** — breathing thinking indicator, collapsible reasoning panel, web-search tool call card, cascading source chips, inline citation superscripts, word-by-word streaming text with markdown.
- **Multi-state chat input** — auto-grow textarea, attach + paste + window-wide drag-drop, image previews, tool toggle chips (Web / Research / Think), model picker, animated send/stop transition.
- **Atmospheric details** — film grain overlay, vignette glow, shimmer text, breathing dots, rotating compass icon, blurred fade-in tokens.

## Try the demo

Click "Compare frontier AI models" on the empty state — or just send any message — and watch the full sequence: thinking → reasoning → searching → reading sources → synthesizing → writing.

## Stack

Next.js 16 · React 19 · Tailwind 3 · Framer Motion · Lucide icons.
